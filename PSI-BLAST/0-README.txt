This directory contains supplementary files for Biochem 3390. 

Query was 6KFS, DB was refseq

The files in the sequence NCBI BLAST ... from QUERY to 8 are the initial setup and the 8 PSI-blast iterations

seqdump-4.txt is the fasta file of the 165 hits

0BN3H81S016-PSSM_Scoremat.asn is the resulting PSSM for further use

COBALT ... is the phylogenetic tree of hits

NCBI Multiple ... is the graphic MSA view of the hitis
